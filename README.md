### 시즌규정

1. 한 시즌의 기간은 7일이며, 7일마다 총 포인트 랭킹을 결산해 보상을 발송합니다.
2. 시즌 포인트 결산 시간: 일요일 밤 23시 정각
3. 시즌 포인트 초기화 시간 : 원요일 5시 정각
4. 시즌 보상 발송 방식: 개인 랭킹 보상은 우편으로 발송되며, 길드 랭킹 보상은 창고로 발송되어 길드장이 분배합니다.

1. 참가 레벨 : Lv.40
2. 길드 KVM 오픈일 : 화/목/일
3. 길드 KVM 오픈시간 :  21:30~23:00
